# 코인 라이프 프로젝트
- 2024년 01월 09일 ~ 2024년 01월 12일 (4일간) 진행판 프로젝트입니다.
- 가상화폐(비트코인) 사용자들을 위한 자유게시판, 질문게시판 기능과 코인차트를 만들었습니다.

## 사용 언어(라이브러리)

### 프론트엔드
- HTML, CSS, JS
- REACT

### 백엔드
- Node,js, Express
- MySQL
